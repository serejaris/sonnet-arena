# sonnet-arena

Browser multiplayer 3D arena shooter (Quake-clone) — Three.js client + Colyseus authoritative server, deployed on a corp-server LXC container. Built fast for a live stream demo: one link, shared room, everyone plays together.

Full plan, architecture, and contracts (network protocol, HTTP, asset pipeline, deployment, testing): see [`PLAN.md`](./PLAN.md).
